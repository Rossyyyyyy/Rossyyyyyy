# Assets

This folder contains auto-generated and custom static assets for the profile README.

| File | Description | Updated |
|------|-------------|---------|
| `header.svg` | Custom animated game-style header banner | Auto (GitHub Action) |
| `stats-card.svg` | Dynamic stats card with live GitHub data | Auto (GitHub Action) |
| `pixel-divider.svg` | Custom pixel-art divider line | Static |

> `header.svg` and `stats-card.svg` are regenerated automatically by the `generate-assets` workflow every 12 hours and on every push to `main`.
